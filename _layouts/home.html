Projects

JMatrix is a lightweight Java package that is designed for working with 2-dimensional matrices efficiently. It involves parallel computations for Numerical calculations using the concept of multithreading. The current implementation supports operations for all the wrapper classes that extend the Number class in java and uses bounded generics to enforce type check during compile time. The implementation also takes into account spatial locality to avoid cache misses during operations involving matrices.

Scalargrad is a reverse automatic differentiation system that supports arithmetic operations like addition, subtraction, exponentiation, multiplication, and division on scalar values. These operations are recorded internally using a directed acyclic graph and later used in computing the derivatives with respect to the weights during backpropagation.

Computer vision contains implementations of various neural network architectures like k nearest neighbors, convnets for image classification from scratch using Python.

Wordgen is a character-level language model that predicts the next character given a sequence of characters as input using a Multi-layer perceptron.

MarkovGenerator is a stochastic model based on the technique of the Markov chain that predicts new names based on the text sample provided. It also has an interactive UI where the user can import the text file based on which the words have to be generated.

Projects

Scalar Autograd engine is an autograd engine that supports arithmetic operations like addition, subtraction, exponentiation, multiplication, and division on scalar values. These operations are recorded using a directed acyclic graph and later used in computing the gradients during backprop.

Image Classification contains implementations of networks like Convnet for Image classification from scratch using Python.

Wordgen is a language model that predicts the next word in a given sequence.

MarkovGenerator is a stochastic model based on the technique of the Markov chain that predicts new names based on the text sample provided. It also has an interactive UI where the user can import the text file based on which the words have to be generated.

Thanks to Andrej Karpathy for his Zero to Hero playlist . Some of the projects were implemented after going through the video series. I highly recommend this playlist for anyone looking to start with deep learning.
